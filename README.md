# ESCSP
 Customized Software for the Eclipse Soundscapes: Citizen Science Project including:
ES WAVES (audio extraction and validation suite) processes and validates AudioMoth recordings,
AME (AudioMoth Metadata Extractor) automates metadata generation,
EPTT (Eclipse Phase Timing Tool) identifies local eclipse phases from NASA predictions,
AZUS (Automated Zenodo Upload Software) (Coming in 2026) enables large-scale dataset publication to Zenodo
