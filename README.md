# ESCSP
 The Eclipse Soundscapes: Citizen Science Project produced suites of specialized software to carry out various tasks related to investigating the effects of the 2023 Annular and 2024 Total Solar Eclipses on the behavior of wildlife.  All software produced by the Eclipse Soundscapes: Citizen Science Project is covered by a BSD-3 license.

The software created for this project includes:

# [Eclipse Look-Up Tool (2023)](https://github.com/ARISA-Lab-LLC/ESCSP-Eclipse-Look-Up-Tool)
A web-based tool that provides eclipse timing and coverage based on ZIP code or location. It simplifies complex NASA data into user-friendly outputs. Updated through the 2024 eclipse; minimal updates are needed for future use.

# [Eclipse Phase Timing Tool (EPTT) (2023)](https://github.com/ARISA-Lab-LLC/ESCSP-Eclipse-Phase-Timing-Tool)
A utility that calculates eclipse phase timings (start, maximum, end) for given locations using NASA datasets. It accepts CSV inputs and returns structured timing data. Designed to support eclipse-related scientific analysis; current through 2024 eclipse; minimal updates are needed for future use.

# [ES WAV Audio Validation & Extraction Suite (ES WAVES) (2024)](https://github.com/ARISA-Lab-LLC/ESCSP-ES-WAV-Audio-Validation-Extraction-Suite)
A Python module that can automatically ingest the data of large numbers of microSD cards, processes and sort large WAV files by serial number into sites based on ES ID 

# [AudioMoth Metadata Extractor (AME) (2024)](https://github.com/ARISA-Lab-LLC/ESCSP-ES-AudioMoth-Metadata-Extractor)
A tool for extracting embedded metadata from AudioMoth recordings, including device serial number, firmware version, temperature data, and precise file timing. This tool aids in verifying data integrity and linking recordings to participant-submitted metadata, processes large WAV files by splitting them into standardized chunks, identifies overlapping recordings across multiple days, and prepares data for time-aligned spectral analysis.

# [AZUS: Automated Zenodo Upload Software (2025)](https://github.com/ARISA-Lab-LLC/ESCSP-AZUS)
A Python-based tool that automates the batch upload of validated datasets and metadata to Zenodo. AZUS ensures each dataset is assigned a DOI and includes properly formatted titles, tags, licensing, and funding information, dramatically reducing manual time-entry tasks and errors, and streamlining the public access process.
